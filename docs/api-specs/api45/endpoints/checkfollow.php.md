# Check follow

**Route:** `GET /checkfollow.php`
**API:** `twitter-api45.p.rapidapi.com`
**Description:** This endpoint get latest subscriptins of the user and latest followers for the target account. And checks if user follows the needed account.  WARNING: might not be suitable for big accounts or old subscriptions.

## Parameters

| Name | Type | Required | Example |
|------|------|----------|---------|
| user | STRING | Yes | - |
| follows | STRING | Yes | - |
